01 / 08 What I Needed to Learn before writing a single line
The Goal
Build a JSON essay explorer — a research tool to view AI-generated essays from 4 models across 3 assignments, show prompts and responses side by side, and display color-coded rubric scores.

I had zero coding experience. I needed to learn how to describe what I wanted precisely enough that Claude could build it for me — and I could understand what it built.
learn How does a 3-panel layout work? Left = file list. Center = content. Right = scores. I sketched this on paper first.
learn What is JSON? It is a structured file that stores my essay data — model name, prompt, response, scores — all in one place.
learn What is GitHub? It stores my code online. I can load my JSON files directly from there into the dashboard.
The System I Was Building
ChatGPT · Gemini · Claude · Grok generate essays → essays stored as JSON files on GitHub → dashboard loads them → shows prompt, response, rubric, scores — all in one view. The dashboard is the final interface for my PhD research.

What I wrote to Claude
Left panel — "Load from GitHub" button · assignment filters: Catlove Case Study, Social Spending, Real Estate · 4 model groups: ChatGPT, Gemini, Claude, Grok · each with Generation, Tuning, Reflection buttons

learn The more specific the prompt, the more precise the code. I described layout, colors, interactions, and rules — Claude read it as a full system spec.
learn Writing the spec made me think like a designer before touching any code. I had to know what I wanted before Claude could build it.

07 / 08 Right Panel — Score Table black bg · 5 rubric dims · color coded
What I specified
Score table on black background · "Scores" header in teal · model name + total score per row · 5 dimensions indented 30px below each model · 4 = green · 3 = yellow · 2 = orange · 1 = red · no visible table borders
parser.js — score to color js
function scoreClass(score) {
  if (score === 4) return "s-capstone";
  if (score === 3) return "s-milestone3";
  if (score === 2) return "s-milestone2";
  return "s-benchmark";
}
/* CSS class drives the color */
